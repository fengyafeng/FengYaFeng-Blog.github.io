PHP手机上传图片更改头像

发表于 2017-02-27 | 分类于 jquery,php |

手机端php上传图片

这里给大家介绍一款非常实用的JQuery图片上传预览插件uploadView，这款插件不仅可以兼容手机和PC，而且代买非常的简介明了。本人就是在项目中，用这款插件来做头像的更改上传的。

1.HTML:

这是JQuery的一个插件，所以在使用前，先引入jquery.min.js，这里就引入你项目的jquery.min.js地址就可以啦。然后就是提交地址了，在这里需要注意的是，在form表单提交时，要加入enctype=”multipart/form-data”。img:src就是预览图片的地址，在未上传时，这里显示之前的图片，所以要从后台分配过来原来的图片地址。

<script src="__PUBLIC__/js/jquery.min.js"></script>
<form action="{:U(GROUP_NAME. '/Setting/run_setting_account')}" enctype="multipart/form-data" method="post">
<div class="btn-upload">
    <a href="javascript:void(0);">
        <!--图片预览 -->
        <div class="js_showBox">
            <img class="js_logoBox" src="{$userinfo['cover']}" width="50" height="50">
        </div>
      </a>
      <input class="js_upFile" type="file" name="cover">
</div>
</form>

2.CSS:

这里是重写了上传按钮的样式，由于我这个案例是做的修改头像，所以直接就把原来的头像作为了按钮，css可以按照自己的需求自己重写。

<style>
/* 上传插件按钮样式，可以自己重写 */
.btn-upload {
  width: 50px;
  height: 50px;
  position: relative;
  margin-bottom: 10px;
}
.btn-upload a {
  display: block;
  width: 50px;
  text-align: center;
  color: #4c4c4c;
  background: #fff;
  border: 1px solid #cecece;
}
.btn-upload input {
  width: 50px;
  height: 50px;
  position: absolute;
  left: 0px;
  top: 0px;
  z-index: 1;
  filter: alpha(opacity=0);
  -moz-opacity: 0;
  opacity: 0;
  cursor: pointer;
}
</style>

3.javaScript:

JS部分没有什么，就是在页面写参数配置之前，先引入jquery.uploadView.js这个插件就好了。参数后面有说明。

<script type="text/javascript" src="__PUBLIC__/js/jquery.uploadView.js"></script>
<script>
    $(".js_upFile").uploadView({
        uploadBox: '.js_uploadBox',//设置上传框容器
        showBox : '.js_showBox',//设置显示预览图片的容器
        width : 50, //预览图片的宽度，单位px
        height : 50, //预览图片的高度，单位px
        allowType: ["gif", "jpeg", "jpg", "bmp", "png"], //允许上传图片的类型
        maxSize :2, //允许上传图片的最大尺寸，单位M
        success:function(e){
            $(".js_uploadText").text('更改');
            //上传成功后，反馈的信息，可根据需求进行更改
        }
    });
</script>

最后就是php方法处理前端页面提交上来的图片，然后根据自己的需求，自行处理。

前端限制form表单多次提交

发表于 2017-02-12 | 分类于 javascript |

问题描述

今天做东西的时候，由于网络卡，在提交表单数据的时候，多点击了几下，结果发现数据库添加了多条同样的数据

解决办法

在form表单提交之后，给button按钮追加一个’disabled‘的元素，让它呈现不可点击的状态，这样就只能点击一次了。

$(function () {
    $('form').submit(function() {
        $('button').attr('disabled','disabled');
    });
})
